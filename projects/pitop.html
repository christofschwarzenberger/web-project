Pi-Top Emotion Detection

Human Robot Interaction

❮ ❯

Overview

As part of the Human Robot Interaction lecture, a Raspberry Pi pi-top robot was programmed, which can recognize faces and detect emotions with the help of a camera. The background of the lecture was to implement social interactions between humans and robots.

The code for the emotion recognition was split into small parts for a student workshop. The goal of the workshop was to introduce the programming language Python and the field of study Computer Science to the participants.

My Role

I was responsible for developing the code modules in Python and for preparing the workshop. In addition, I prepared a concept for the students, which gradually evolves from simple code to control the robot motors, towards face recognition and emotion detection.
